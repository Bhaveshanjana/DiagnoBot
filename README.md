# DiagnoBot 

**DiagnoBot** is an intelligent symptom analysis web application designed to help users get educational insights into their possible health conditions. By entering their symptoms and personal health data, users receive possible conditions, recommended actions, and trusted health resources ‚Äî all powered by RapidAPI.
